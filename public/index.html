Swiss Federal Legislation — All Acts & Ordinances

REST API Reference

All endpoints return JSON. Rate limits: 30 req/min (standard), 10 req/min (text fetch).

Search
GET /api/search?q=keyword&lang=de&limit=20&in_force=true
List all
GET /api/list?lang=de&limit=50&offset=0&in_force=true
Law metadata
GET /api/legislation/:sr_number?lang=de
Example: /api/legislation/101?lang=fr
Full text
GET /api/text/:sr_number?lang=de&format=html&version=2024-01-01
format: html (plain text) | xml (Akoma Ntoso)
Historical versions
GET /api/versions/:sr_number
Citation network
GET /api/citations/:sr_number?direction=both&lang=de&limit=50
direction: from | to | both
MCP SSE endpoint (for AI assistants)
GET /sse  →  SSE stream
POST /messages?sessionId=id  →  send MCP messages
Health
GET /health
